# Objects

### Each object can hold many properties specified as follows

```javascript
let person = {
    firstName: `John`,
    lastName: `Luke`
  };
    //Two ways to output the objects and its properties
    console.log(`${`Hi ` + person.firstName} ${person.lastName}`);
    console.log(`Hi ${person.firstName} ${person.lastName}` );
```
